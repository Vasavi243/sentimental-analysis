# 💬 Sentiment Analysis Web App

<p align="center">
  <b>Analyze the sentiment of text using Machine Learning 🚀</b><br>
  Built with Flask + NLP + Modern UI
</p>

---

## 🌐 Live Demo

👉 https://sentimental-analysis-cggd.onrender.com

## 📌 Features

✨ Real-time sentiment prediction
📊 Confidence score & probability bars
🧠 Machine Learning model (TF-IDF + Naive Bayes)
📚 Example tweets for testing
🕘 Local prediction history
🌐 Fully deployed web app

## ⚙️ Installation & Setup

### 1️⃣ Clone the repo

```bash
git clone https://github.com/Vasavi243/sentimental-analysis.git
cd sentiment-analysis-app
```

### 2️⃣ Install dependencies

```bash
pip install -r backend/requirements.txt
```

### 3️⃣ Run the app

```bash
python backend/app.py
```

### 4️⃣ Open in browser

```
http://localhost:5000
```
